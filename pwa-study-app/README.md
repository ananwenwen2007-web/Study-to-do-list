# 学习助手 PWA

初一学生自主学习 To-Do List 应用，支持任务管理、朗读打卡、英语听写、积分系统。

## 功能

- **任务管理**：添加/完成/删除每日学习任务，支持普通、朗读、听写三种类型
- **拍照提交**：完成任务后拍照上传作为凭证
- **朗读打卡**：录音 + 语音识别，自动判断朗读完整度
- **英语听写**：TTS 朗读单词，自动批改，正确率 >= 80% 发放积分
- **积分系统**：累计积分、今日积分、历史记录
- **教材内置**：译林版七年级英语、人教版七年级语文、必背古文诗歌

## 部署到 Vercel（2 分钟）

### 方法一：拖拽上传（最简单）

1. 打开 [vercel.com](https://vercel.com)
2. 用邮箱或 GitHub 注册/登录（免费）
3. 点击 **"New Project"**
4. 将 `pwa-study-app` 整个文件夹拖拽到上传区域
5. 点击 **"Deploy"**
6. 等待 30 秒，获得你的专属链接（如 `https://your-app.vercel.app`）

### 方法二：通过 GitHub 部署（推荐，方便更新）

1. 在 GitHub 创建新仓库
2. 将 `pwa-study-app` 文件夹内容推送到仓库
3. 在 Vercel 点击 **"New Project"** → **"Import Git Repository"**
4. 选择你的仓库，点击 Deploy
5. 以后更新代码推送到 GitHub，Vercel 会自动重新部署

## 使用方式

### 电脑浏览器
1. 打开部署后的链接
2. 建议收藏到书签栏

### iPad / iPhone
1. 用 Safari 打开部署后的链接
2. 点击底部 **分享按钮**（方框+箭头）
3. 选择 **"添加到主屏幕"**
4. 桌面会出现应用图标，点击即可使用

### 添加到主屏幕后
- 应用会全屏显示，没有浏览器地址栏
- 像原生 APP 一样使用
- 数据保存在设备本地

## 技术说明

- 纯前端应用，无需后端服务器
- 数据存储在浏览器 localStorage
- 语音合成使用 Web Speech API（浏览器自带）
- 语音识别使用 SpeechRecognition API（Chrome/Safari 支持）
- 支持 PWA，可离线使用
